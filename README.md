# Eqxia Invoice Generator

Formulaire web pour générer des factures PDF professionnelles avec le branding Eqxia Ltd.

## Fonctionnalités

- Formulaire interactif avec tous les champs nécessaires
- Calcul automatique de la Due Date selon les conditions de paiement
- Ajout/suppression de lignes de prestations
- Calcul automatique des sous-totaux, taxes et totaux
- Génération PDF 2 pages directement dans le navigateur (via jsPDF)
- Page 1 : Détails facture, prestations, totaux
- Page 2 : Coordonnées bancaires MCB + CGV

## Stack

- React 18 + Vite
- jsPDF (génération PDF côté client)
- Aucun backend nécessaire

## Développement

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Déploiement

Connecter le repo GitHub à Vercel. Aucune configuration supplémentaire nécessaire.
